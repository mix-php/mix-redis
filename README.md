## Mix Redis

redis for use in multiple execution environments, with support for FPM, CLI, Swoole, WorkerMan, and optional
connection pool (coroutine)

可在各种环境中使用的 redis，支持 FPM、CLI、Swoole、WorkerMan，可选的连接池 (协程)
